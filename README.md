# ♻️ Municipal Waste Management Intelligence System

### AI/ML based decision support system for optimizing waste collection, recycling efficiency, and sustainable urban management.

---

## 🚀 Overview

This project is an **AI-powered municipal waste analytics system** that predicts waste management cost and provides actionable insights for urban decision-makers.

It integrates:

* Machine Learning cost prediction
* KPI dashboard visualization
* AI-based insights for optimization


## 📊 Key Features

* 📈 **Cost Prediction (₹/Ton)**
* 📊 **KPI Dashboard**

  * Efficiency Score
  * Recycling Rate
* 🧠 **AI Insights**

  * Cost optimization suggestions
* 🏙️ **Municipal Decision Support**
* 🎯 Clean and modern UI (Streamlit)


## 🧠 Machine Learning Model

* Algorithm: Linear Regression / Pipeline Model
* Input Features:

  * Waste Generated (tons)
  * Recycling Rate (%)
  * Population Density
  * Collection Efficiency
  * Awareness Campaign Score


## 🖥️ Tech Stack

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn

## 📌 Sample Output

* Predicted Cost: ₹5200+ per ton
* Efficiency Score: 7
* Recycling Rate: 60%
* Insight: Improve recycling to reduce cost
